List vs Dict (easy words)

Use `list` when:
- You have a collection of items in order.
- You want to access by position (index): `items[0]`, `items[1]`.
- The data is like a sequence (scores, names, numbers in order).

Use `dict` when:
- You have values you want to find using a key (name/word/id).
- You want to look up like `d["name"]` or `d["word"]`.
- The data is like “key -> value” (student_name -> student info, word -> count).
